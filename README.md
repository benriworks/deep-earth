# 地球地下シミュレータ (Earth Underground Simulator)

地球の内部構造を科学的に正確かつ直感的に体験できる教育用 3D Web アプリケーション。
[要件定義書](./earth-underground-sim_requirements_v1.1.md) v1.1 に基づいて実装。

## 機能

- **3D 地球レンダリング** — マウス/タッチで回転・ズーム・パン(OrbitControls)
- **断面カットアウェイ** — ハーフ/クォーターカットで内部の層構造を露出、カット角度も変更可能
- **レイヤー構造表示** — 地殻・上部マントル・遷移層・下部マントル・外核・内核を正しい半径比で表示。層ごとの表示切替・不透明度調整・選択して物性値(深度/密度/温度/地震波速度)を確認
- **地震波シミュレーション(屈折対応)** — 震源(深さ・位置)を設定して P 波・S 波の伝播を再生。波線は速度勾配で湾曲し、層境界ではスネルの法則で屈折・反射する。再生/一時停止/スクラブ/速度調整対応
- **観測点と走時記録** — 断面円周上の観測点が到達順に変色し、走時表と走時曲線(震央距離-到達時刻)を表示。地震学の一次データそのもの
- **シャドウゾーンの定量化** — S 波が届かない領域(本モデル約93°以遠、実地球約103°)と P 波シャドウ帯(約91〜118°、実地球約103〜143°)を角度つきで断面上にハイライト。「届かない波」から液体外核を推論した科学史を体験できる
- **深度-物性プロファイル** — 密度・温度・地震波速度の深度グラフ。モホ面・グーテンベルク面・レーマン面の不連続と、外核で Vs=0 になる様子が一目で分かる。プローブ現在深度と連動
- **ガイドツアー** — 「人類はどうやって地球の中身を知ったのか」を8ステップで辿る解説モード(1906〜1936年の発見史つき)
- **クイズモード** — 体験した内容を7問で確かめる(シャドウゾーンの推論・発見史・波の屈折など。解説つき)
- **地殻の誇張表示** — 実スケールでは見えない地殻(半径比0.5%)を見かけだけ厚く表示するトグル。誇張していること自体を明示
- **マントル対流** — 対流セルの上昇流(赤)・下降流(青)をパーティクルで可視化
- **仮想プローブ** — 地表の任意地点から地心へ降下し、現在深度の層と物性値をリアルタイム表示

## 技術スタック

| 分類 | 技術 |
|---|---|
| フレームワーク | Next.js 15 (App Router) + TypeScript (strict) |
| 3D | Three.js + @react-three/fiber + @react-three/drei |
| 状態管理 | Zustand |
| UI | Tailwind CSS + shadcn/ui |
| テスト | Vitest |

## 開発

Node.js 20+ と pnpm が必要。

```bash
pnpm install
pnpm dev        # 開発サーバー (http://localhost:3000)
pnpm test       # ユニットテスト
pnpm lint       # ESLint
pnpm build      # 本番ビルド
```

## デプロイ

Vercel を想定。GitHub 連携またはローカルから:

```bash
pnpm dlx vercel deploy
```

## 設計メモ

- **60FPS 維持**: 毎フレーム変化する値(プローブ深度・波面時刻)は React state に載せず、`useFrame` 内で Zustand の `getState()` を直接参照。UI への反映は約 10Hz に間引き
- **断面カット**: `THREE.Plane` の clippingPlanes(クォーターは 2 枚 + `clipIntersection`)+ 層ごとの円環キャップメッシュで切り口を表現
- **地震波**: 震源設定時に波線追跡を事前計算してテーブル化(Web Worker 不要の軽さ)。波線方程式 `d(dir)/ds = −(v′/v)(r̂ − (r̂·dir)dir)` を中点法でデカルト積分し、層境界はベクトル形式のスネルの法則(全反射は反射継続)。レイパラメータ p = r·sinθ/v の保存をテストで検証。シャドウゾーン境界の解像度確保のため CMB 臨界射出角近傍に追加レイを配置
- **科学データ**: PREM(予備的基準地球モデル)の境界値を層内線形補間した近似。教育目的の簡略化・誇張(プローブ降下速度、対流速度など)は UI 上で注記

## 科学的な簡略化について

本シミュレータは教育目的であり、以下の簡略化を含みます:

- 層内の物性値は PREM 境界値の線形補間(実際の PREM は深度の多項式)。このため S 波シャドウは約93°から始まり、実地球の約103°より小さめに出る(実 PREM は最下部マントルで速度勾配が平坦になり、波線がより遠くまで届くため)
- 波種変換(SKS など)や表面反射(PP など)は扱わない。S 波は核マントル境界で消滅、地表に達した波線はそこで終了
- プローブの降下速度・マントル対流の速度は視覚化のため大幅に誇張
- 地殻は実スケールでは半径比 0.5% と薄く、図ではほぼ見えない
